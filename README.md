# Maven Market -Interactive Performance Report
**Project Overview:**\
Maven Market is a fictional multi-national grocery chain operating in Canada, Mexico, and the United States. The goal of this project was to create an interactive Power BI report to analyze company performance and highlight key insights for decision-making.

**Tools & Technologies:**
  - Microsoft Power BI
  - DAX for custom calculations

**Steps:**
1. **Data Preparation:** Connected and cleaned source data for accuracy.
2. **Relational Modeling:** Built relationships between tables to ensure reliable reporting.
3. **Metrics & Calculations:** Developed calculated columns and measures using DAX, including Total Revenue, Profits, Orders, and Return Rate.
4. **Report Design:** Designed an interactive dashboard using Cards, Line Charts, Bar Charts, Tree Map, Decomposition Tree, and Gauges to visualize key metrics.

**Key findings:**
- **Highest Profit:** Hermanos generated $21,753 in 1998.
- **Top Sales in December:** San Andres – 1,347 sales; Portland – 1,000 sales.
- **Highest Return Rate:** Product brand Horatio – 1.26% in 1998.

**Outcome:**
The interactive dashboard provided actionable insights into regional performance, sales trends, and product returns, demonstrating the ability to convert complex datasets into understandable reports for decision-making.
